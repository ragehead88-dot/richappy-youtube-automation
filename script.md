# 리채피 × bibl 공통 대본 골격 (script.md)

> 분석 기반: richappy 상위 영상 3편 (16,373 / 8,196 / 322회) + bibl 상위 영상 4편 (36,036 / 11,943 / 10,949 / 4,314회)
> 작성일: 2026-06-25

---

## 핵심 발견: 두 채널의 공통 흥행 공식

| 구간 | richappy 패턴 | bibl 패턴 | 공통 원리 |
|------|--------------|----------|---------|
| 0~15초 | 시청자 고통 묘사 | 초보자 실수 지적 | **나의 이야기처럼 느끼게** |
| 15~60초 | "진짜 이유는 따로 있다" | "이것만 바꾸면 달라진다" | **기존 상식 뒤집기** |
| 1~2분 | 결론 먼저 + 유명인 권위 | 숫자 예고 + 직접 경험 | **신뢰 + 호기심 동시 자극** |
| 2~10분 | 3~5막 구조, 비유·스토리 | 단계별 실전 가이드 | **설명이 아닌 여정** |
| 마지막 | 심리·내면 연결 + 격려 | 비즈니스 연결 + 실행 촉구 | **감정 마무리** |

---

## 대본 골격 (5단계 템플릿)

### 1막 — 훅 (0:00~1:00) ★★★ 가장 중요

**목적:** 시청자가 "나 얘기잖아"라고 느끼게 만들기

```
[고통 묘사형]
"(시청자의 현실 상황)을 하고 있는데 왜 (원하는 결과)가 안 나올까요?
이건 (일반적 원인)이 부족해서일까요? 아닙니다."

[충격 사실형]
"(놀라운 사실/숫자). 그런데 (반전 상황)이 일어나고 있습니다."

[비교 대조형]
"비슷한 (조건)인데 어떤 사람은 (성과A), 어떤 사람은 (성과B)입니다.
차이는 딱 하나였습니다."
```

**richappy 실제 오프닝 패턴:**
- "주식 유튜브 다 챙겨보고 밤 설쳐가며 공부하는데 왜 계좌는 제자리일까요? 이건 정보가 부족해서일까요? 아닙니다."
- "저와 함께 투자를 시작한 동료들. 비슷한 공부, 비슷한 기업인데 결과는 완전히 달랐습니다."

**bibl 실제 오프닝 패턴:**
- "처음 영상을 올리고도 조회수가 오르지 않을 때 대부분 초보 유튜버는 '소질이 없나 보다'며 포기합니다. 근데 같은 영상이라도 이 세팅만 바꾸면 조회수는 완전히 달라질 수 있는데요."
- "열심히 해서 올리더라도 평소 100회, 200회밖에 안 나온다면 가장 먼저 신경 써야 되는 부분이 바로 썸네일과 제목입니다."

---

### 2막 — 결론 먼저 + 예고 (1:00~2:00) ★★

**목적:** 떠나지 않을 이유 제공 + 영상 구조 제시

```
"결론부터 말씀드리겠습니다. (핵심 답 1~3줄).

오늘 영상을 끝까지 보시면
- (얻는 것 1)
- (얻는 것 2)  
- (얻는 것 3)
을 알게 되실 겁니다.

그럼 왜 이런 이야기를 드리게 되었는지, 하나씩 살펴보겠습니다."
```

**핵심 포인트:**
- "결론부터"는 richappy의 시그니처 — 신뢰 구축
- bibl은 "숫자 예고"로 대체 ("15가지", "3가지 방법")
- 영상 끝까지 볼 이유를 명시적으로 말함

---

### 3막 — 권위/근거 제시 (2:00~4:00) ★★

**목적:** "왜 이 사람 말을 믿어야 하는가" 해결

**richappy 방식 — 유명인 인용:**
```
"흥미로운 건 제가 말하는 이 주장이 단순히 저만의 관찰이 아니었습니다.
(유명인 + 직책/규모)도 정확히 같은 이야기를 했거든요.
(유명인)은 (비유/사례)를 들었는데요..."
```
- 켄 피셔 (560조 운용) + 복싱 선수 비유
- 워런 버핏 + "카지노가 붙은 교회" 표현
- 하워드 막스 + 두 가지 조심할 태도

**bibl 방식 — 직접 경험:**
```
"제가 운영하는 이 채널들에 모두 적용해 둔 세팅들입니다.
실제로 (결과)가 나왔고요."
```

**공통 원리:** 추상 주장 → 구체 사례/비유로 반드시 착지

---

### 4막 — 전개 (4:00~끝-2분) ★

**목적:** 약속한 내용 전달, 이탈 방지

**구조 원칙:**
1. 숫자로 분류 ("첫 번째", "두 번째") — 진행 상황 인지 가능하게
2. 각 항목 = [문제/현상] → [원인] → [해법] 순서
3. 중간 비유 1개 필수 (카페 비유, 복싱 비유 등)
4. 항목 사이 전환: "그런데 여기서 중요한 게 있습니다" / "마지막으로"

**richappy 추가 요소 — 심리 연결:**
```
"이건 우리가 (이유)가 아니라 (심리적 원인) 때문입니다.
우리 뇌는 (원리 설명)..."
```

**bibl 추가 요소 — 실전 가이드:**
```
"설정 방법은 이렇습니다. (유튜브 스튜디오 접속 → 단계별 클릭)
잘 모르겠다면 두 가지 방법을 활용해 보세요."
```

---

### 5막 — 클로징 (마지막 1~2분) ★★

**목적:** 감정 마무리 + 행동 촉구 + 구독 유도

**richappy 시그니처 클로징:**
```
"오늘 말씀드린 (핵심 1), (핵심 2), (핵심 3).
이 세 가지를 알고 투자하는 것과 모르고 투자하는 것의 성과는
시간이 갈수록 무섭게 벌어질 겁니다.

여러분은 반드시 행복한 부자가 됩니다."
```

**bibl 시그니처 클로징:**
```
"이걸 알고 유튜브를 시작하면 시간, 멘탈, 체력을 아낄 수 있습니다.
오늘 말씀드린 것 중 하나라도 바로 적용해 보세요."
```

**공통 요소:**
- 핵심 내용 1~2문장으로 압축 요약
- 행동 촉구 (지금 바로 할 것)
- 감정적 격려 or 미래 비전 제시
- 구독/좋아요 요청은 가볍게 or 생략

## 후킹 위치 & 전환 포인트

| 시점 | 기법 | 예시 |
|------|------|------|
| 0초 | 고통/충격으로 시작 | "왜 계좌는 제자리일까요?" |
| 30초 | 반전 ("진짜 이유") | "이건 정보가 부족해서가 아닙니다" |
| 1분 | 결론 먼저 제시 | "결론부터 말씀드리겠습니다" |
| 2분 | 권위 인용 투입 | "560조 운용가도 같은 말을 했습니다" |
| 중간 | 비유로 환기 | "마치 복싱 선수처럼..." |
| 끝-2분 | 심리/감정 연결 | "이건 지식의 문제가 아닌 마음의 문제" |
| 끝 | 격려 + 행동 | "여러분은 반드시..." |

## 제목 공식 (실측 기반)

### 대박 공식 (richappy 기준)
```
[유명인 이름] + [핵심 키워드] + [숫자/결과]
예: "560조 자산 운용가 켄 피셔가 인정한 투자 방법" → 16,373회
예: "워런 버핏과 하워드 막스가 본 현재 주식 시장" → 8,196회
```

### 대박 공식 (bibl 기준)
```
[숫자] + [행동] + [결과]
예: "유튜브 조회수 미친 듯이 터지는 15가지 채널 설정" → 36,036회
```

### 중박 공식
```
[역설/반전] + [진짜 이유]
예: "유튜브를 그만두는건 열정이 부족해서가 아닙니다" → 4,329회
예: "결핍의 마음이 판단을 망칩니다" → 322회 (유명인 없이 심리 축만으로)
```

## 채널별 차이점 (적용 시 주의)

| | richappy | bibl |
|---|----------|------|
| 주제 | 투자 심리·마음·철학 | 유튜브 실전 성장법 |
| 톤 | 따뜻한 조언자, 경험담 중심 | 실용적 강사, 단계별 가이드 |
| 권위 근거 | 유명 투자자 인용 (필수) | 직접 운영 채널 실적 |
| 전개 방식 | 심리→철학→감정 연결 | 실전 가이드→비즈니스 확장 |
| 클로징 | "행복한 부자가 됩니다" 격려 | 지금 바로 실행 촉구 |
| 숫자 공식 | 3가지, 5가지 심리/마음 | 15가지, 3단계 실전법 |

## 리채피 대본 작성 시 체크리스트

- [ ] 오프닝 15초 안에 시청자 고통 or 충격 사실 제시
- [ ] 30초 이내 "진짜 이유는 따로 있다" 반전 제시
- [ ] 1분 이내 "결론부터" + 영상 끝까지 볼 이유 명시
- [ ] 유명 투자자/전문가 최소 1인 인용 (이름+직책+규모)
- [ ] 핵심 개념에 비유 1개 이상 (추상→구체)
- [ ] 전개는 숫자 구조 (3가지 or 5가지)
- [ ] 심리/마음 축으로 귀결하는 구간 포함
- [ ] 클로징: 핵심 요약 + "여러분은 반드시 행복한 부자가 됩니다"
- [ ] 전체 길이: 10~15분 (richappy 채널 기준 최적)
