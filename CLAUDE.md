## 유튜브 커뮤니티 게시글 구조
① 훅(2~3줄): 수치 대비·역설·깨진 법칙으로 시작
② 공감+상황(2~3줄): 독자의 불안/혼란 짚기
③ 통념 제시 → "아닙니다." 단호하게 반박
④ 진짜 원인 + 전문가/연구 인용 (투자 대가 이름 + 구체적 수치)
⑤ 개인 경험 2~3줄 (실패 → 깨달음)
⑥ 행동 제안 2~3가지
⑦ 독자 참여 유도 ("여러분은 어떻게 생각하시나요?")
⑧ 시그니처: "여러분은 반드시 행복한 부자가 됩니다."
- 길이: 600~900자 / 단락 최대 3~4줄 / 이모지·해시태그 없음
- 금지: 5줄 이상 단락, 추상적 표현, 첫 줄 자기소개

## Threads 게시글 구조
① 첫 줄: 수치 대비형 / 질문형 / 역설 선언형 / 사실 직격형 중 택1
② 상황 묘사 2~3줄
③ "사람들은 ~라고 생각합니다. / 아닙니다. / 실제로는 ~입니다." 구조
④ 근거/수치/인용 2~3줄
⑤ 핵심 통찰 단독 1줄 강조
⑥ 행동 유도 질문 또는 다음 편 암시
- 시리즈: [1/5]~[5/5] 형식, 시그니처는 마지막 편에만
- 길이: 400~700자 / 문장 10~20자 짧게 / 이모지·해시태그 없음
- 어미 비율: -죠(35%) / -ㅂ니다(30%) / -요(20%) / -다(15%)
